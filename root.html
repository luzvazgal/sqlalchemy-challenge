  • /api/v1.0/tobs: Returns a JSON list of temperature observations (TOBS) for the previous year

  • /api/v1.0//: Return a JSON list of the minimum temperature, the average temperature, and the max temperature for a given start-end range.
    Calculate the TMIN, TAVG, and TMAX for dates between the start and end date inclusive
